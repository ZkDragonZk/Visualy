O que é Pendrive Bootavel?

Precisamos primeiro saber que um pendrive ele funciona da mesma forma de um cd de instalação de sistema qualquer. para inicializar o pc a diferença que eh possivel colocar um sistema operacional qualquer (com imagem iso) ele nos permite da boot ao sistema para que podemos usar outros sistemas operacionais. lembre-se que ele vai apagar tudo que tem dentro do pendrive então lembre-se de fazer um backup antes!.

Usando o Woe-Usb [Metedo 01]

Quando eu começei usar o linux pela primeira vez foi uma batalha para mim volta para o windows, então aprendi a criar o Pendrive Bootavel pelo o famoso Woe-USB foi bem fácil de utilizar apenas usando algumas linhas de comando então vamos lá! ele funcionara tambem para Ubuntu e Xubuntu.

ㅤㅤㅤ

$ Primeiro quero você abra seu lindo terminal

$ sudo apt install git p7zip-full python3-wxgtk4.0

$ sudo pip3 install Woe-USB-ng

ㅤㅤㅤ

Pronto Instalado! agora funciona mesma coisa o que o nosso lindo Rufus vamos lá! segue abaixo a explicação, e lembre-se que as imagem ficaram disponivel aqui!.

ㅤㅤㅤ

$ Primeiro quero que você abra seu Woe-USB

$ Agora insira seu pendrive

$ Agora aonde está escrito from a disk image (iso) seleciona sua iso

$ Agora aonde está escrito target device seleciona seu Pendrive

$ Agora você clicla aonde está escrito install para instalar a iso

ㅤㅤㅤ

Viu? Foi bem fácil e rapido! agora caso não conseguiu fazer esse metado tente o proximo metado abaixo! também e super fácil é rapido! Vamos lá!

Usando o Eatcher [Metedo 02]

Bem minha historia com o Eatcher é bem loka, conheci ele usando o ARCH linux, quando o Woe-USB não conseguia instalar no ARCH linux, mas ele não uma coisa muito complexa por final e bem fácil de ser utilizado apenas dando 1 clique já está tudo já bootavel! o que beleza então vamos lá!.

ㅤㅤㅤ

$ Primeiro acesse esse site

$ Agora desça ate a parte Get your assets e faça o download para seu sistema linux 64x ou 32x

$ Faça a instalação normal!

ㅤㅤㅤ

Bem se você chegou ate aqui meus parabéns continua assim que você vai conseguir! Vamos lá lembre-se as imagem estão aqui!

ㅤㅤㅤ

$ Agora quero que você abre seu Eatcher

$ Agora insira seu pendrive

$ Agora onde está escrito select image você vai selecionar sua iso

$ Agora no lado do botão select image vc seleciona seu pendrive. mas ele identifica sozinho as vezes

$ Pronto agora e so você dar um Finish
